# 🏪 Enunciado 1 — Sistema de Inventario de Tienda
## Título: Sistema de Gestión de Inventario con Persistencia de Datos (CSV)
*Caso de uso:* Como empleado de una tienda, necesitas un sistema para gestionar el inventario de productos.
Datos del producto: 
1. ID
2. nombre
3. categoría
4. precio
5. cantidad en stock.

 El sistema debe permitir:

1. Agregar nuevo producto (ID único) check
2. Listar todos los productos check
3. Buscar producto por ID o nombre  check
4. Actualizar precio o cantidad check (proteger id)
5. Eliminar producto con id check (puedo hacerlo por nombre también)
6. Extra: alertar cuando un producto tenga stock menor a 5 unidades (mañana) listo

> Requisitos técnicos: listas, diccionarios, funciones, try/except, persistencia en CSV.
